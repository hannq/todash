“Utils” 方法


_.Array2Union<array>

将 数组/元组 转化为 union


参数


array (Array): 需要处理的 数组/元组 类型


返回


union (Union): 返回由 数组/元组转换而成的 联合类型(union)。


例子


type UnionFromArray = _.Array2Union<['vue', 'react', 'angular']>;
// => 'vue' | 'react' | 'angular'

type UnionFromArray = _.Array2Union<('vue' | 'react')[]>;
// => 'vue' | 'react'
